Adding/Removing Passengers to A Carpool
 
Adding/Removing Passengers
 
You have the ability to add or remove your children from a carpool event. By tapping on an event block, the Event Details screen appears. To add passengers
 
1) Tap the +/- Passengers icon on the bottom menu.
 
2) On the following page tap passenger(s) to add or remove them from the passengers list. Tap again to deselect them. A check mark will appear next to the selected passenger’s names.
 
3) Tap “Done”
 
If your event is a repeating event a pop up will appear asking, For This Event Only, For This and Future Events, or For All Events.
 
Viewing Passengers for an Event
 
To view the passengers for an event:
 
1) Tap the event block.
 
The lists of passengers to and from the event are listed on the Event Details page.
